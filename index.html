🔄 Set up household sync

Connect to Firebase so you and your spouse see the same inventory in real time — free, no subscription.

  • Go to console.firebase.google.com and sign in with your Google account (not the Advanced Protection one — use any Google account).
  • Click Create a project, name it anything (e.g. "home-inventory"), disable Google Analytics, click Create project.
  • In the left sidebar click Build → Firestore Database → Create database → choose Start in test mode → pick any location → Enable.
  • In the left sidebar click the gear icon ⚙ → Project settings. Scroll down to "Your apps", click the Web icon </>, name it "inventory", click Register app. You'll see a firebaseConfig block — copy the values below.

Your spouse opens the same URL on their phone, goes to this screen, and enters the same API Key and Project ID. That's it — both phones sync instantly.
